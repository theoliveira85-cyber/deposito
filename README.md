# Dashboard de Estoque - Produtos de Limpeza

Sistema de gerenciamento de estoque para produtos de limpeza.

## GitHub Pages - Configuração de Deploy

Para configurar o GitHub Pages e fazer o deploy desta aplicação:

1. Acesse as configurações do repositório no GitHub
2. Navegue até **Settings** > **Pages**
3. Em **"Source"**, selecione **"Deploy from a branch"**
4. Em **"Branch"**, escolha **"main"** e pasta **"/ (root)"**
5. Clique em **"Save"**

Após alguns minutos, a aplicação estará disponível em: `https://theoliveira85-cyber.github.io/deposito/`

## Arquivos Principais

- `index.html` - Página inicial que redireciona para deposito.html
- `deposito.html` - Aplicação principal do dashboard
- `manifest.json` - Manifesto PWA para instalação como app
- `service-worker.js` - Service worker para funcionalidade offline
